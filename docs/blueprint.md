# **App Name**: JasTip Express

## Core Features:

- Order Submission: Allow users to submit their JasTip (jasa titip) orders, including item descriptions, quantities, and specific requests.
- Order Tracking: Enable users to track the status of their orders in real-time, from placement to completion.
- Formatted Nota Generation: Generate well-formatted 'nota' (receipt) for each order, ready for printing, with clear itemization and total cost.
- Payment Processing: Integrate a secure payment gateway to allow users to pay for their orders online.
- Product Description Refinement: Refines user-submitted product descriptions to ensure clarity and accuracy for JasTip service providers.

## Style Guidelines:

- Primary color: A vibrant orange (#FF7043), evoking energy and excitement, crucial for an express delivery service.
- Background color: Light peach (#FEEBC8) creates a soft and inviting backdrop that enhances readability and user comfort.
- Accent color: A lively cyan (#00BCD4), provides visual contrast for call-to-action buttons and interactive elements.
- Body and headline font: 'PT Sans' (sans-serif), blends modern clarity with approachability.
- Use clean, modern icons to represent various order stages and product categories.
- Prioritize a clear, intuitive layout that emphasizes key order details and tracking information.
- Employ subtle animations to provide feedback on user interactions and enhance the sense of progress.
